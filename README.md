### Built With

- HTML
- CSS
- Javascript
- ReactJS
- React Hooks (useState, useEffect) 

## Features

- Add/remove notes
- Light/dark mode toggle
- Notes saved locally
- Cross platform
- Tablet & mobile friendly

### Run locally 

Clone the project

```bash
  git clone https://github.com/trstefan/react-notes.git
```

Go to the project directory

```bash
  cd react-notes
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
