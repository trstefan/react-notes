## 📝 Project Description

Sticky Notes application built using React and React Hooks.

## 📚 Technologies

- ReactJS
- ReactHooks
- CSS

## 🎯 Features

- Add/remove note
- Dark Mode
- Notes saved locally
- Cross platform
- Tablet & mobile friendly

## :runner: Run Locally 
Clone the project

```bash
  git clone https://github.com/trstefan/react-notes.git
```

Go to the project directory

```bash
  cd react-notes
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
